# Proyecto-RPA-Automatización-de-presentaciones-con-Claude
mini-RPA que toma el resultado de un prompt y lo transforma automáticamente a PPTX o HTML. Herramientas: Claude (para generar/normalizar el contenido), la funcionalidad Visual/vision de Claude para leer PDFs/imágenes si hace falta, y Python para la orquestación y conversión.
